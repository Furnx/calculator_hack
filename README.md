# Hackathon: Git Calculator (120 Mins)
Mission: Collaborate on a Python calculator using professional Git workflows.

# Prerequisites

-Git & Python: Installed.

-GitHub: Account ready.

-Setup: Fork the https://github.com/keammakola/calc-hack and git clone to your machine.

# Phase 1: Project Management (30 Mins)
Milestones: Create a v1.0-Basic-Ops milestone in GitHub.

Issues: Create tasks (e.g., "Add Addition," "Add Subtraction").

Assign: Each member claims an issue and links it to the milestone.

# Phase 2: Development Loop (60 Mins)
Branch: create a branch for each feature

Code: Write your function in calculator.py.

Commit: make sure your commit messages are detailed

Push

PR: Open a Pull Request to the original repo.

# Phase 3: Integration & Finalize (30 Mins)
Merge: Host reviews and merges PRs.

Conflicts: Fix markers (<<<<, ====, >>>>) if two people edit the same line.

README: Update with team names and usage instructions.

Sync: Everyone runs git pull origin main.

Test: Run python calculator.py.
